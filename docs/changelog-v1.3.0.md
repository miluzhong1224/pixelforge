# PixelForge V1.3.0 变更日志

> **版本**: v1.2.1 → v1.3.0 | **日期**: 2026-06-24 | **作者**: 产品经理

---

## 一、版本概述

V1.3.0 新增「反向生图」模式，支持上传图片 → AI 反推 Prompt 或直接生成风格变体。

---

## 二、新增功能

### 2.1 反向生图（F2）

生成页新增第三个模式 Tab：「反向生图」。

**反推提示词**：
- 上传图片 → Qwen3-Omni-30B-A3B-Captioner 视觉模型分析
- 自动生成中文 Prompt，填入输入框
- 切换到文生图模式即可基于该 Prompt 创作

**生成风格变体**：
- 上传图片 → 直接调用图生图 API
- 生成与上传图风格相似的变体图片

---

## 三、文件变更

### 新增
- `src/app/api/prompt/reverse/route.ts` — 反推提示词 API（调用 VL 视觉模型）

### 修改
- `src/app/(app)/generate/page.tsx` — 新增「反向生图」Tab + 反推提示词/生成变体双按钮
