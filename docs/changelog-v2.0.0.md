# PixelForge V2.0.0 变更日志

> **版本**: v1.5.0 → v2.0.0 | **日期**: 2026-06-26 | **作者**: 产品经理

---

## 一、版本概述

V2.0.0 为全站 UI 重构版本。色彩系统从暗紫切换到黑白蓝极简风格，亮色背景替代暗色背景。

---

## 二、设计变更

### 2.1 色彩系统

| 元素 | V1.x (暗紫) | V2.0 (黑白蓝) |
|------|------------|--------------|
| 页面背景 | `#0a0a0f` | `#fafafa` |
| 卡片 | `#131320` | `#ffffff` |
| 主文字 | `#f0f0f5` | `#0d0d0d` |
| 辅助文字 | `#8888a0` | `#666666` |
| 主色 | `#7c3aed` 紫 | `#0066ff` 蓝 |
| 边框 | `#27272a` | `#e5e5e5` |
| 输入背景 | `#1a1a2e` | `#f5f5f5` |

### 2.2 风格方向

- 从暗色 AI 工具感 → 亮色极简专业工具感
- 参考: Linear / Vercel / Apple 设计语言
- 保持功能不变，纯视觉升级

> 30+ 文件全站颜色替换。
