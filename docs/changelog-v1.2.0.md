# PixelForge V1.2.0 变更日志

> **版本**: v1.1.1 → v1.2.0 | **日期**: 2026-06-22 | **作者**: 产品经理

---

## 一、版本概述

V1.2.0 面向"从能用到好用"，新增新手引导、操作进度反馈、UI 细节优化。

---

## 二、新增功能

### 2.1 新手引导（F7）

首次进入生成页面自动弹出 3 步引导，帮助新用户快速了解操作流程：

1. **输入描述词** — 介绍了 Prompt 输入和 AI 优化翻译
2. **选择风格模板** — 引导用户浏览模板库
3. **开始生成** — 介绍了尺寸预设和生成按钮

特性：
- 首次访问自动弹出，看完后不再显示（localStorage 记录）
- 支持「跳过引导」和「上一步/下一步」导航
- 进度条指示当前所处步骤

### 2.2 操作进度反馈（F11）

去背景和放大功能增加了实时进度指示：

**编辑页**：标题栏显示处理中状态
```
🔄 正在去除背景...  12s
🔍 正在放大增强...   8s
```
按钮同步显示 loading 状态

### 2.3 模板按钮放大

Prompt 输入区域下方的模板入口从右上角小字改为独立一行全宽按钮：
- 虚线边框 + hover 高亮效果
- 文字：「浏览 Prompt 模板库（12 套风格预设）」
