# PixelForge V1.5.0 变更日志

> **版本**: v1.4.0 → v1.5.0 | **日期**: 2026-06-26 | **作者**: 产品经理

---

## 一、版本概述

V1.5.0 聚焦**社区互动**：新增评论、个人主页、广场搜索、点赞优化，以及「我的」页面整合。

---

## 二、新增功能

### 2.1 社区评论
- 广场作品卡片悬浮 → 点击 💬 图标 → 弹窗评论
- 自动使用登录账户昵称，无需手动输入
- localStorage 持久化

### 2.2 个人主页
- 广场点击用户名 → `/user/:id` 个人页
- 展示头像、昵称、公开作品数、作品墙
- 支持一键复用 Prompt

### 2.3 广场搜索
- 探索页顶部搜索框
- 按 Prompt 关键词 / 用户名实时过滤

### 2.4 点赞优化
- 每人每作品只能点赞一次（localStorage 记录）
- 刷新后已赞状态保持

### 2.5 「我的」页面整合
- 导航「我的作品」→ 重命名「我的」
- 新增个人信息卡片（头像/昵称/邮箱/作品数）
- 新增「🌐 已发布」筛选标签
- 去重「新建创作」按钮
- 标签重排：全部 → 文生图 → 图生图 → 局部重绘 → 已发布 → 收藏

---

## 三、文件变更

### 新增
- `src/app/(app)/user/[id]/page.tsx`
- `src/components/community/comment-modal.tsx`

### 修改
- `src/app/(app)/dashboard/page.tsx` — 个人信息卡 + 已发布标签 + 按钮去重
- `src/app/(app)/explore/page.tsx` — 搜索框 + 评论入口 + 点赞限制 + 用户名可点击
- `src/app/(app)/layout.tsx` — 导航「我的作品」→「我的」
